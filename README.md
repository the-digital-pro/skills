# Skills

My agent skills that I use every day to do real engineering - not vibe coding.

**Current release:** [`v0.2.1`](https://github.com/the-digital-pro/skills/releases/tag/v0.2.1)

## Quickstart (30-second setup)

1. Run the skills.sh installer:

   ```bash
   # latest (tracks main)
   npx skills@latest add the-digital-pro/skills

   # or pin to a specific release
   npx skills@latest add the-digital-pro/skills@v0.2.1
   ```

2. Pick the skills you want, and which coding agents you want to install them on.

3. You're ready to go.
